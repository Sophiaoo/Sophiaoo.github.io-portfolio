Introduction

HTML stands for Hypertext Markup Language. It allows the user to create and structure sections, paragraphs, headings, links, and blockquotes for web pages and applications. HTML is not a programming language, meaning it doesn't have the ability to create dynamic functionality.

Cascading Style Sheets is a style sheet language used for describing the presentation of a document written in a markup language such as HTML. CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.

How HTML works

Each HTML page consists of a set of tags (also called elements), which you can refer to as the building blocks of web pages. They create a hierarchy that structures the content into sections, paragraphs, headings, and other content blocks.

Most HTML elements have an opening and a closing that use the syntax. For example;

  • Tag for a header <h/>- the opening, and </h/>- the closing.

    • for example, to make a H1 heading

    <h1>

    Hello World

    <h1/>
  • To make a word, letter or sentence bold/italic, we use the <b></b>, and <em></em> respectively.

    • for example

    <b>Hello World</b>
    <em>Hello World</em>
  • There is also a table tag for creating tables.

    • for example

    <table>
    <tr>
    <th>Firstname</th>
    <th>Lastname</th>
    <th>Age</th>
    </tr>
    <tr>
    <td>Jill</td>
    <td>Smith</td>
    <td>50</td>
    </tr>
    <tr>
    <td>Eve</td>
    <td>Jackson</td>
    <td>94</td>
    </tr>
    </table>

The output will look like this on the right. With 'Tr' meaning table rows, 'th' table heading, and 'td' table column.

Firstname Lastname Age
Jill Smith 50
Eve Jackson 94

Most Used HTML Tag

HTML tags have two main types:

  1. Block-Level Tags: elements take up the full available space and always start a new line in the document. Headings and paragraphs are a great example of block tags.
  2. Inline Tags: only take up as much space as they need and donâ€™t start a new line on the page. They usually serve to format the inner contents of block-level elements. Links and emphasized strings are good examples of inline tags.

What is CSS?

CSS Introduction

  • CSS stands for Cascading Style Sheets
  • CSS describes how HTML elements are to be displayed on screen, paper, or in media
  • CSS svaes a lot of work. It can control the layout of multiple web pages at once
  • External stylesheets are stored in CSS files

Why use CSS?

Css is used to define styles for your web pages, including the design, layout and variations in display for different devices and screen sizes. Example;

body { background-color: lightblue; } h1 { color: white; text-align: center; } p { font-family: verdana; font-size: 20px; }

The CSS style applied to the body in the example, makes the background color white, the H1 headings white and in the center, all paragraphs in verdana font, with a 20px size. Controlling multiple web pages at the same time.

Why use HTML and CSS together?

While HTML is used as the foundation for every web page, CSS is what gives it a display very easy on the eyes and makes interactive.

Both of them make the foundation of everything you'll need to build your website.
